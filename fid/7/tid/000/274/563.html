涵涵是个17岁的女高中生  她的学习在学校里已经是非常条件的存在了  同时受着许多男生的喜欢  和同性朋友的嫉妒
---------- 8 ： 0 0
  ：啊啊迟到了都8点了！妈我早餐就不吃了！
  ：早餐很重要的  涵涵你至少喝一杯温开水再出去！
  (涵涵匆忙的喝了一大杯600的温开水就出门了)
---------- 上 学 路 i n g
  涵涵在上学的路上突然被一阵尿意刺激了一下  迫不得已只能在大马路上用手捂住不让尿液喷出  此时她突然想起昨晚为了美容喝了1000+的水  然而早上没有释放晨尿还喝了一大杯开水  想到这里涵涵的膀胱就一阵难受  
  os：斯～好胀好想上厕所…啊…要漏了…呜好想舒服的泄洪…
  (此时涵涵看到了一处正在排队的女厕所)
  ：啊啊啊有厕所了！但是在排队…好想尿尿  但快要迟到了呜
  (最后涵涵犹豫了一番还是快走去厕所排队了)
  ：你好小姐姐我快憋不住了可以让我先嘛
  ：抱歉我也很急  你去找别人吧
  ：拜托了小姐姐我已经漏了再不去真的要喷出来了
  ：你憋了那么多让你先不是得尿很久吗？起开我要尿尿！(用力的把涵涵推开)
  由于涵涵体力不支的关系所以一把坐在了地上  涵涵死死的捂住尿道口不让尿出来  感受着膀胱的酸胀感  又听着厕所里震耳欲聋的哗哗声和舒服的呻吟  此时眼泪已经不争气的掉了下来实在太难受了  涵涵也知道自己等不到隔间就去上学了  
---------- 学 校 の 考 试
  涵涵夹着腿终于到了学校  进教室时已经开始考试了  只好坐在位置上继续憋着  写着写着涵涵的膀胱终于受不了这种压力了  鼓起勇气去讲台上和监考老师报告
  ：老师我可以去趟厕所吗真的忍不住了…
  ：你上学迟到现在又要中途去厕所？回去考试！
  涵涵也只好回到位置上死死忍着  不好的事这一幕正好被班上的一群心机女目睹了  她们看着眼眶泛红肚子非常鼓的涵涵心里萌生了不好的想法…
---------- 学 校 の下 课
  涵涵的膀胱已经接近极限了  所以只好一步一步挪去厕所  好不容易到了厕所却被心机女璐璐们堵在扫具间里
  ：你们要干嘛？快让开我要去尿尿！
  ：搞笑了  我们今天就非不想让你去怎么了？有本事抢校草就有本事把你的尿给我忍着！
  ：我没有！是他自己来找我的！你们快点走开我要喷出来了…
  ：很急吗？我们今天就是要让你在校草面前失禁～(勿模仿
  (璐璐狠狠的踢了涵涵肚子一脚 涵涵被踢得直流眼泪 )
  ：你今天小心点  我们是不会让你去尿的～（摁了一下涵涵的膀胱就走了
---------- 学 校 の 上 课
  这节是涵涵最喜欢的音乐课  但还是被强烈的尿意干扰的心烦意乱  加上又听着老师播放的抒情歌曲(水声  涵涵肚子里的水更加难忍了…好不容易「欣赏」玩抒情音乐就要开始上台考试了  看着一个个同学上去却没有心情欣赏  本优美的歌曲现在也变成了噪音。终于轮到了涵涵上台  此时的她已经直不起身子  突然站起来的感觉也给尿道口增添了许多压力  实在忍不回去漏出来一些  有几滴清澈的水珠透过内裤从白皙的大腿上滑落  还好不明显没有多少人看见  涵涵一开口声音都在颤抖考试也没有很高分  但她顾不了那么多了  赶紧回位置坐下隔着裙子捂的紧紧的  
---------- 学 校 の 下 课
  涵涵期待很久的下课终于到了  但有了上节课的经验她已经不敢贸然去厕所了  只好偷偷摸摸的进厕所  但无奈她的速度太慢还是晚了璐璐他们  当涵涵刚成功进入隔间就被璐璐拉了出来  
  ：跟你说过了今天都不会让妳尿  我说到做到喔～
  ：你们真的够了！我连晨尿都没放你们还不让我去！
  ：懒得跟你废话那么多  来人给他灌水！
  璐璐给涵涵灌了大概有600+了此时涵涵已经精疲力竭的坐在地上任由他们嘲讽  一心只想把这些如洪水般的尿全都释放出来  但只要失禁的话就会被璐璐那些人拍下来发到网路上  为了面子涵涵还是拼命忍住
---------- 学 校 の 上 课
  涵涵还是一样膀胱酸胀的不行  轻轻一摸像个小孕妇  面对老师的提问她也不吭声  同时还要被同桌干扰  一直讲许多笑话让脸色不好的涵涵笑出来  最终  同桌的搞笑话术还是让涵涵笑出来了  一下子 涵涵的裙子上应了一大块湿斑  她实在受不了对着同桌大喊「我想上厕所忍不住了不要在刺激我了！」这一下全班都安静了  受惊吓的同桌缓过来急忙道歉  老师也不敢再耽误时间  尽快的让大家下课了
---------- 学 校 の 午 餐
  到了中午涵涵想着只要吃完饭就可以摆脱璐璐他们了  于是大口大口的把餐盘里的饭菜都一下子吃完了  就连鱼汤也一滴不剩  看着空荡蕩的餐盘涵涵虽然很撑很胀  但想到即将可以去厕所泄洪瞬亮眼一亮  捂住小腹就小跑步奔向厕所  但不好的事发生了  好不容易跑到厕所门口却发现璐璐等人已经早早的在厕所门口埋伏好了  涵涵还在厕所门口焦急的想办法  突然灵机一动 拿着门口的桶子就往小树林狂奔  也不管流到腿上的尿液了  到了小树林找好没人的地方就掀开裙子开始狂喷  看着自己别了6个小时多的尿现在正如大瀑布般喷出来  涵涵也舒服的忍不住开始呻吟 听着自己憋了很久的尿释放的哗哗声  涵涵心里还是很羞耻的  大概狂喷了2分多钟她才尿完  2000毫升的桶子被尿的满满的
