P1-安心容量  P2-典型容量  P3-实用容量 P4-极限容量

P1 ----  xxxx ml  临界点P1是从完全没有尿意的状态像稍微能感觉到肚子里有尿的状态的一个临界点。所以又称安心容量点P1。

P2 ----  xxxx ml  临界点P2是从完全是指从从S1之后到尿意不可忽略，跨入心慌的状态的一个点，一般人会选择这个时候去放（不玩憋得时候）。 所以又称典型容量点P2。

P3 ---- xxxx ml  临界点P3指虽然心慌，但是还很有信心憋住。这个时候即使没有厕所的存在，也绝对没有尿裤子得危险。在这个范围内，同好还可以自由活动，不至于行为太不自然 。 但超过这个数值就会影响正常活动工作走路，比如用手帮忙。所以也称实用容量点P3。

P4 ---- xxxx ml 从P3继续憋上去，直到极度坐立不安（失禁）， 或者肚子疼 （疼痛）， 或者麻木（偷漏）或者憋到没有尿意（出不来）的情况。我们暂且把它定义为极限容量点P4。

P5 等您来补充。

最后关于憋尿的训练，在保证身心健康的基础上，提升P3意义最为重大。因为这个是很实用的。一味的提升P4虽然数字可能好看，但是一点都不实用，因为在室内和室外憋尿玩全不是一回事。我有个好友她能憋着XXXX cc在室内洗碗，但是如果在外面憋着那个数值一定很惨。
