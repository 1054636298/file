这个憋尿测试关注的是你的“憋尿能力”
定义为 ①憋尿越多 ②时间越长 那么“憋尿能力”就越大

其中
“憋尿越多”用膀胱里累积的尿量(毫升)作为定量依据 (注意不单单是喝下去的水量 而是实实在在在膀胱里的尿量
“时间越长”用憋尿的时间(小时)作为定量依据
在游戏结束的时候
自己计算尿量与时间的乘积 (单位是毫升×小时)
数字越大就越厉害

例如一个人如果憋1000毫升的尿坚持1小时 那么它的憋尿能力就是1000毫升×小时
这个人就跟憋500毫升的尿坚持2小时 (憋尿能力也是1000毫升×小时) 相同

具体玩法如下
游戏初始时排空并开始计时
接下来每个小时内均匀分几次喝水
自己确定每个小时内打算喝多少水
不过每小时至少需要喝下的水按照下面的公式根据下面的公式计算

男生每小时至少需要喝下的水(毫升) = 0.5686 x (公斤体重^0.425) x (厘米身高^0.725) / (年龄^0.203) + 103.5 x (摄氏度气温/25)^3.322
女生每小时至少需要喝下的水(毫升) = 0.4220 x (公斤体重^0.425) x (厘米身高^0.725) / (年龄^0.203) + 86.5 x (摄氏度气温/25)^3.322
(注意: 符号^表示乘方，例如100^2=100 x 100)

例如 一个体重为70公斤 身高为175厘米的25岁男孩子 在一个不开空调的35°C的房间里玩这个憋尿游戏
那么每小时至少需要喝下393ml的水
这样才能保证肾脏有充足的水分源源不断地产生尿液
低于这个数值的话 身体会因为出汗及蒸发散失的水分导致尿液不断减少

每个小时想喝多少水就喝多少水 随你
自己想憋多久就憋多久 随你
期间你可以正常工作 学习 上课 逛街等等
不用总想着憋尿 (特别适合开学了上课时使用

游戏结束的时候 用下面的公式计算膀胱内储存尿量随时间的乘积 也就是你的“憋尿能力”啦

1. 如果憋尿时间少于2小时，则用下面的公式：

男生的数值 (毫升×小时) = 0.2843 x (公斤体重^0.425) x (厘米身高^0.725) / (年龄^0.203) x (累计憋尿的小时^2)

女生的数值 (毫升×小时) = 0.2110 x (公斤体重^0.425) x (厘米身高^0.725) / (年龄^0.203) x (累计憋尿的小时^2)
  
2. 如果憋尿时间多于2小时，需要考虑膀胱充盈对于尿液产生速率的非线性抑制作用，则用下面的公式

男生的数值 (毫升×小时) = 0.2843 x (公斤体重^0.425) x (厘米身高^0.725) / (年龄^0.203) x (累计憋尿的小时^2) x [(每小时实际喝水毫升数/每小时至少需要喝下的水)^0.625]

女生的数值 (毫升×小时) = 0.2110 x (公斤体重^0.425) x (厘米身高^0.725) / (年龄^0.203) x (累计憋尿的小时^2) x [(每小时实际喝水毫升数/每小时至少需要喝下的水)^0.714]

这个数值越大越厉害

不好意思楼主，还有一个问题，就是，这个每小时喝水量是一定要均匀到每小时的吗？还是事后算平均值即可？因 ...

喝水量最好是每个小时都比较相近
因为从数学上来说 憋尿能力(单位为毫升×小时)应该是膀胱中尿量随时间的积分
但是因为运算太复杂
我这里只考虑了每个小时喝水量都一样的情况 简化了计算
如果每个小时喝水量不一样的话 这几个公式算出来的结果可能有误差

此外 不太建议用这个公式计算睡觉的时候的憋尿能力
一方面睡觉后长时间没有喝水 出汗及蒸发散失的水分会导致分泌尿量没有达到最大值
另一方面夜间抗利尿激素分泌会增多促进尿的重吸收
因此尿的生成速率会大大降低
也就是说 喝同样的水
可能白天能憋5个小时 夜里就可以憋8个小时了 (这两个数字只是打比方用 不代表真正的比例关系
看起来仿佛憋尿能力增强了呢

无论快喝还是慢喝
只要超过了产尿所需要的最低饮水量
每分钟膀胱的产尿速率就基本是一个变化不大的定值了
也就是说膀胱充盈的速度基本是一个常量
(当然当膀胱达到过度充盈的时候 胃肠道吸收会降低 产尿量则会降低 这个是憋到比较难受的时候才发生的 这个暂时不讨论)

讨论一个简单的理想状态
假设憋尿4个小时(240分钟) 在憋尿【结束的时候】释出量为1450ml 并且假设没有达到过度充盈的话(例如你的膀胱容量很大
那么可以假设膀胱里的尿量基本是线性增加的
也就是说第60分钟时尿量是312ml 第90分钟时尿量是543ml 第120分钟时尿量是725ml
假设某个人到500ml才有比较强烈的尿意的话 那么其实真正难受是90分钟之后才开始的
也就是只难受了150分钟

而如果假设憋尿8个小时(480分钟) 在【憋尿结束的时候】释出量为1450ml 并且假设没有达到过度充盈的话(例如你的膀胱容量很大
那么第120分钟时尿量是312ml 第180分钟时尿量是543ml 第240分钟时尿量是725ml
假设某个人到500ml才有比较强烈的尿意的话 那么其实真正难受是180分钟之后才开始的
也就是难受了300分钟

所以其实慢憋会比快憋要更具有挑战性
也就是为什么慢憋会出来的分数高

当然必须承认的是 “难受”的程度无法定量描述
快憋和慢憋的难受程度可能并不一样
并且有的人可能喜欢快憋 有的人喜欢慢憋
所以目前很难量化呢
这里只是假设一旦超过了某个阈值 “难受”的程度就是一样的了
未必是准确的呢
以后如果我会量化难受的程度了 可以改进这个游戏做出第二版来
期待一下呀

此外 身材瘦小的人的肾小球过滤速率会小一些
因为一个人的器官(例如 肾)血流量(因为尿是靠过滤血流产生的)大概跟体重成1/2到3/4次方的比例关系 (这个叫做Allometry理论
而膀胱的充盈与否本身与血流量没有什么关系 其膨胀程度与体重几乎没有关系
所以身材瘦小的人其实更容易憋得更久
