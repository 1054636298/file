經驗分享 非愛好者的網路調教

文章長 我稍微分一下段落

一、以前經驗談

關於找一個憋尿一起玩得妹子這種事我本來已經放棄一陣子
但最近開始各種嘗試 策略是這樣
總結以前失敗原因
是因為每次都想說先不暴露愛好
慢慢聊等到差不多在開始嶄露自己的癖好
但發現很多妹子就算聊得再好 只要說有這個癖好大概就是兩條路 封鎖 和 慢慢不回很難真的找到想要的人
於是我調整策略，開始先從遊戲開始試探 畢竟人都是愛玩的 因此我制定了幾種遊戲

1. 猜拳 =>
輸了喝水 贏了可以讓輸家做一項任務 (還有許多細項以下省略)
如果妹子可以接受 在用這樣子不刺激我們加一條規則 三個小時不上廁所 妳可以嗎 之類的
如果妹子不能就放棄找下一個
如果可以再往下追加規則

2. 大富翁 =>
做了個線上大富翁程式 如果採到某處就執行任務 或 選擇喝水(一樣先玩)
再加規則 如果中間抽到廁所卡才能去
如果妹子問幹嘛這樣 可以先退 說想比比看 還是妳不敢之類的 如果不接受就放棄
大概是這樣循序往下加規則

關於遊戲 類似不一一舉例子
但這些遊戲過程中也會找到妹子的一些奇怪屬性 需要一點耐心尋找
塞瓶子/斗m/喜歡被稱讚/需要陪/想要棒棒...等等
當成獎勵慢慢開發 在完成一次漸漸滿足對方不要一次急著稱讚或全給
慢慢往下 開始要求 照片 => 錄音  => 視訊

二、關於癖好

以前的我總覺得這是一個羞於讓大家知道的癖好甚至是缺點，
以至於每次提出時好像都是女生施捨才願意玩這個愛好，導致常常失敗
但經過許久發現 愛好其實在交友軟體是強項 原因有下面

1. 總有沒話講得時候 => 這時候可以當話題調戲一下對方 或用開玩笑口吻帶過
2. 一定是共通話題 => 很少有妹子沒有忍耐經驗，沒有一定是你引導得不好或太無聊
3. 吸引眼球 => 如果你下一般標題 或 一般聊天 其實很快會被當一般人，因此即使不是愛好，也應該有特長才好聊，如果沒有，
剛好愛好就是你的特長可以增加聊天趣味，但不要一直聊 要間斷 要插話 要聊生活
大概想到三點還是實際看各位狀況應用這樣

三、妹子情況

妹子比較年輕，個性跟之前分享的差有點多，可能是沒有經驗的關係
年紀大概高一(16)而已，因此玩之前,我還是有滿滿的罪惡感，畢竟我都是27.8的年紀了，沒找過這麼嫩的妹子 XD
這個妹子line上只有一個 "琦" 外號"狗狗" 聊到她最慘經驗是在在活動中喝1000cc忍耐六小時
但當初從她的經驗談就知道可以有開發潛力，目前也還在慢慢嘗試中


四、今天憋的情況

妹子
3:00 先喝900cc的 牛奶 和 飲料
4;00 去一次廁所
6;00 很想去

於是給妹子兩種選擇
1. 馬上放掉 但等等補水900cc到12點
2. 繼續忍耐 但可以到10點結束 放水錄影片
妹子當然選擇2

6:30 再讓他喝下700cc手搖杯一杯

但是到8點快不行了 => 於是妹子居然偷偷放200cc
我讓她補水400cc 並開始做任務

到10點的時候 本來要讓妹子放尿
視訊直播尿尿聲 和 尿尿
但是妹子死活不願意露私處 因此我也不好強迫 只要求看尿柱 和聲音
但在馬桶上發現無論如何都無法成功避開
20分鐘後 家人過來差點被發現 於是妹子忍住，沖水離開廁所

到了11點多 妹子快撐不住了 用跪姿，坐姿，正躺，緩解尿意，但家人還沒睡怕被發現
於是硬是忍到12點  感覺過去了 但還是很脹很滿
但此刻他家人在洗澡
最後到1點去解放
但之前尿在馬桶遮不住私處，因此決定說服妹子尿在地板上(說服超久)


最後附上 玩的照片幾張 不過很不清楚就是了 但妹子第一次視訊尿尿，很緊張還一直上不出來
最後一泡尿放了快五分鐘 慢慢一下有一下沒有可能是拍影片很緊張會回憋的關係 這讓我想到被禁尿的男友小說中的淑芳
原來女生緊張還真的會尿不出來，即使非常急 也會慢慢一股股出來
