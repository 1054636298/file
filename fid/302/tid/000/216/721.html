论坛有很多憋尿的玩法，十分有趣，不过大多玩法往往是基于一些基本的任务衍生变化出来的，万变不离其宗，在这里，我对一些常见的基础憋尿玩法做了分类整理，以方便那些需要任务指导的小伙伴们按需自取。

关于评分：
根据自己的理解，本人对每一个任务从（难度/刺激度/羞耻度）做出了5分满分的评价，放在每个任务的后边括号内。
其中，难度是指任务是否容易完成；刺激度是指任务对感官上的刺激，包括生理上的兴奋感；羞耻度是指任务对个人心理的冲击程度。
评分相当主观，如果觉得有失偏颇，那么你说的对。

注：
1.        每个评分是对动作的基础评分，例如按压小腹评分为（1/1/1），但可以通过增加按压强度、增加按压次数、提高憋尿量等来提高难度和刺激度。
2.        基础任务之间可以联合，从而让任务不再单调，也可以大幅提升三项评分，例如可以在下蹲的同时按压小腹，也可以在公众场合放松尿道等等。
3.        为了提升任务乐趣，可以尝试搭配上类似于骰子、转盘之类的游戏机制。
4.        欢迎大家讨论进行补充。


===========正文===========

1.        蹂躏小腹
享受小腹酸胀的感觉，可以看这些。

1.1        挤压类
通过按压等方式挤压膀胱，是最基础的憋尿刺激方式，虽然看起来简单，但在尿意十分强烈时按压小腹，也可能带来强烈的刺激，甚至失禁。
#        手掌按压小腹（1/1/1）
#        拍打小腹（1/1.5/1）
#        按压小腹隆起的侧边部位（1/1/1）
#        按压小腹后旋转揉小腹（2/2/1）
#        书本等重物平放在小腹上（1/1.5/1）
#        用绳子勒小腹（2.5/2/1.5）       

1.2        体操类
体操类指用各种身体的动作去压迫膀胱，也是人们脑洞大开的地方，由于各种动作真的花样繁多，因此这里只将常见的动作分类整理，难免挂一漏万，待后续版本进一步补充完整。
静态
#        坐姿（0/1/0）
#        脚尖站立（1.5/1/0.5）
#        平躺（1/1/0.5）
#        开腿平躺（1.5/1/1.5）
#        平趴（2.5/2/1）
#      平趴在书本上（3/3.5/1）
#        下蹲（2/2/1.5）
#        跪坐（1/1/1.5）
#        跪坐前趴（2/2.5/2）
#        鸭子坐（2.5/2.5/1.5）
#        坐位体前屈（3/2/1.5）
#        抱腿坐（2/3/1.5）
#        跪姿向后弯腰（2.5/3/2）
#        下腰（3.5/3/2.5）
#        劈叉（4.5/3.5/3）
#        M开脚（2/2/4）

动态
#        蹲起（3/3/2）
#        高抬腿（4/4/2）
#        慢跑（3.5/3.5/2）
#        仰卧起坐（2/3/1.5）
#        小燕飞（4/4.5/2.5）
#        腹式呼吸（1/2.5/1）
#        跪姿爬行（1.5/1.5/3.5）
#        空中脚踏车（1.5/2.5/1.5）
#        青蛙跳（3/4/3）

1.3        紧身穿着
紧身衣物在憋尿时往往有锦上添花的作用，根据其紧身程度，难度和刺激程度有很大差别，有时为了更强的包裹感，也可以套穿多件，这里仅作简单分类，评分仅作参考。
#        裤袜（1/1.5/1.5）
#        紧身裤（1/1.5/2）
#        束腰带（2/2.5/2.5）

2.        锻炼意志
喜欢憋尿时心理上的兴奋感，可以看这些。

2.1        尿意习惯培养
憋尿不一定要很强烈，长时间若有若无的尿意就像一直有人在挠痒一样，而为了享受这种感觉，则可以通过练习让自己习惯尿意的存在。
#        睡觉憋尿（3/1.5/1.5）
#        憋晨尿（2/2.5/2）
#        憋尿听课、上班（2/2/2.5）
#        排尿不排完（2.5/3/3）

2.2        尿意忍耐
强烈的尿意不一定意味着极限，也可能是自己精神出现了松懈，经常有人说憋过了最强烈的一阵之后反而不那么急了。不妨用这些方式进一步增强尿意，来强化自己对尿意冲击的抵抗能力。
#        在尿意强烈时延迟排尿（3.5/3.5/2.5）
#        在尿意强烈时喝水（1.5/2.5/1）
#        模拟排尿（2.5/3.5/3）
#        听水声（1/2/1）
#        用水冲洗小腹（2/3/2）
#        洗澡（3/4.5/2.5）
#        看他人排尿视频或小说（1/2.5/2.5）

2.3        羞耻感
喜欢憋尿的人也往往会对过程中的羞耻感欲罢不能，而在很多时候，这种羞耻感往往只是自己心理上的一道坎（当然有些真的会社死），而在跨过这种心理上的门槛之后会令人感到愉悦。
#        公众场合憋尿（1.5/1/2）
#        告诉他人自己尿急（2/1.5/3）
#        上课等正式场合报告尿急离场（2.5/1.5/3.5）
#        尿液摆放在开放空间展示（1.5/1/4）
#        憋尿自拍（1.5/1.5/3.5）
#        排尿过程录制（1.5/1.5/4）
#        尿湿衣物（1.5/1.5/3.5）
#        穿尿湿过的衣物（2/1.5/4）
#        尿湿座位、床等（2.5/1.5/3.5）
#        野外排尿（2/1.5/3.5）
#        站姿排尿（1.5/1/2.5）
#        被把尿（1.5/1.5/4.5）
#        排尿时与他人通话（1.5/1/3）

3.        尿道刺激
憋尿的酸胀感不仅来自于膀胱，也来自于尿道，喜欢的话可以看这些。

3.1        回憋
回憋任务十分常见，但变种并不多，因为其实并不需要那么多变种，每一次回憋都会让动作难度上升一分，尿道在疲惫不堪时的回憋往往是决堤的前戏。
#        用力收紧尿道（1/1.5/1）
#        尿尿中途停止（2.5/2.5/2）
#        尿出少量停止（3/3.5/2.5）
#        在快要尿出时停止（3/4/2.5）
#        控制尿道，保持住马上就要尿出的感觉（4/4/2.5）
#        控制尿道，减小排尿水流（2.5/3/2.5）
#        堵住尿道口，做排尿动作（4/4.5/3.5）

3.2        直接刺激
有许多玩具和玩法可以跳过憋尿直接刺激尿道，或许，对于大众而言，直接刺激尿道是比憋尿更加常见的xp也不一定。
#        揉尿道口（1.5/2/2）
#        尿道塞插入（2.5/3.5/3）
#        振动棒震动尿道口（1.5/3/2）
#        震动尿道塞（2.5/4.5/3）
#        尿道扩张（4/4.5/3.5）

4.        特殊玩法
高端玩家们可能已经不屑于一般的玩法了吧。

4.1        圣水
就是直接玩尿液，这方面受众在我观察来看并不多，很多人在憋尿时会有兴奋感，但当实际接触到尿液时则会觉得恶心。不过，xp的事情，谁比谁高级呢？
#        喝自己的尿液（3/1.5/4）
#        喝他人的尿液（4.5/1.5/4.5）
#        尿液涂抹（3/1.5/4）
#        尿液灌肠（3.5/2/3.5）

4.2        导尿管
导尿管真是改变憋尿玩法的东西，可以开发的玩法花样很多，但由于需要插入尿道，因此很多人觉得不安全而排斥。个人的建议是，如果觉得不安全就不要尝试，如果想要尝试，就请做好卫生安全措施。
#        导尿管导出尿液（2/1.5/3）
#        导尿管回灌尿液（3.5/3/3.5）
#        导尿管回灌其他液体（3.5/3/4.5）
#        日常佩戴导尿管及储尿袋（2/2/4）

4.3        失禁
憋到最后总归是要尿出来的，相较于安安静静地尿，有些人更喜欢尿得轰轰烈烈。
#        自然状态下极限失禁（2.5/3.5/3）
#        捆绑极限失禁（3/3.5/4）
#        公众场合极限失禁（3/3.5/4.5）
#        电击失禁（3.5/4/3）
#        高潮失禁（3.5/4/3）
#        喷泉（3/3.5/4）
