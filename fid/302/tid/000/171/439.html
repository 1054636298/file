不仅考验憋尿过程中的意志力，面对满腹尿水和自由支配的排尿机会，你是否还能忍耐？你又是否准备好面对用尽排尿机会却还要饮水的绝望？过程共6-12h
每半小时可以进行一次选择，可以选择三种活动（A排尿 B饮水 C意志力训练）的其中一个进行，不同难度有不同的具体选项可供选择，不同难度存在不同的选择限制。排尿可以计时也可以计量，有容器计量的话可以按照时间排尿，排完后计量，超出规定量两次算作挑战失败。
1.初试憋尿 适合容量：800以上
时长6h，10次选择，6小时结束可以排空。
排空膀胱后喝下300ml水开始挑战。
选项：
  A排尿 共3次 每次7s（200ml）
  B饮水 共5次 2次300ml 3次200ml
  C 意志力训练 共2次
    1次 去厕所脱下内裤坐在马桶上模拟排尿30s
    1次 用力压小腹5次

2.普通的挑战 适合容量：1000以上
时长8h，14次选择，8小时结束可以排空。附加选择限制，每次选择排尿前必须先选择一次饮水再选择一次意志力训练。
排空膀胱后喝下300ml水开始挑战
选项
     A 排尿 共4次 2次5s（150ml） 2次10s（300ml）
     B  饮水 共6次 2次200ml 2次300ml 2次400ml
     C  意志力训练 共4次
         1次 去厕所脱下内裤坐在马桶上模拟排尿1分钟
         1次 用力压小腹10次
         1次  排尿1s停止15s，重复5次。1s内没能停下，多尿1s喝50ml水并延长最后排空时间10min。
         1次  允许排尿0-10s（0-300ml），但每排1s需要排完喝30ml水  

3.不是谁都能完成的挑战 适合容量：1200以上
时长10h，18次选择，10h结束后可以在30min内分2次排空，要求最后一次排出量不能少于800ml。附加选择限制，每次选择排尿前必须先选择一次饮水再选择一次意志力训练。
排空膀胱后喝下400ml水开始挑战。
选项
     A 排尿 共5次 2次3s（100ml） 1次5s（150ml，需要中断1次）2次15s（400ml，需要中断3次）
     B 饮水 共7次 1次 200ml 4次300ml 2次400ml
     C 意志力训练 共6次
        2次 去厕所脱下内裤坐在马桶上模拟排尿2分钟
        1次 用力压小腹15次
        2次  排尿1s停止30s，重复5次。1s内没能停下，多尿1s喝50ml水并在最后一次选择后延长10min结束。
        1次  允许排尿0-10s（0-300ml），但每排1s需要排完喝40ml水

4.真正的意志力挑战 适合容量：1300以上
时长12h，21次选择，12h结束后可以在45min内分3次排空，要求最后一次排出量不能少于1000ml。附加选择限制，每次选择排尿前必须先选择一次饮水再选择一次意志力训练。
排空膀胱后喝下400ml水开始挑战。开始半小时后饮水400ml，开始1小时后开始第一次选择。
选项
     A 排尿 共7次 2次6s（100ml，需要中断2次） 3次9s（150ml，需要中断3次）2次18s（300ml，需要中断5次）
     B 饮水 共7次 1次200ml 3次300ml 3次400ml
     C 意志力训练 共7次
        2次 去厕所脱下内裤坐在马桶上模拟排尿3分钟
        2次  喝下200ml水后，排尿1s停止30s，重复10次。1s内没能停下，多尿1s喝50ml水并在最后一次选择后延长10min结束。
        1次  允许排尿0-10s（0-300ml），但每排1s需要排完立即喝40ml水
        1次 叠2张餐巾纸放在内裤里，控制少量排尿尿湿纸但是不能尿湿内裤。如果在厕所外进行尿湿内裤不需要惩罚，在厕所内进行，尿湿内裤则在最后一次选择后延长20min结束。
        1次 在60s内持续循环排尿与回憋，每次中断至少2s，完成15次奖励排出150ml，少于12次需要饮下300ml水，少于10次需要饮下300ml水并在最后一次选择后延长20min结束。

5.应该完成不了的挑战？ 适合容量：？
时长12h，21次选择，12h结束后可以在45min内分3次排空，要求最后一次排出量不能少于1200ml。附加选择限制，每次选择排尿前必须先选择一次饮水再选择一次意志力训练。
排空膀胱后喝下500ml水开始挑战。开始半小时后饮水400ml，开始1小时后开始第一次选择。
选项
     A 排尿 共7次 3次6s（100ml，需要中断2次） 2次9s（150ml，需要中断3次）2次18s（300ml，需要中断6次）
     B 饮水 共7次 1次200ml 2次300ml 3次400ml 1次500ml
     C 意志力训练 共7次
        2次 去厕所脱下内裤坐在马桶上模拟排尿3分钟
        2次  喝下300ml水后，排尿1s停止30s，重复10次。1s内没能停下，多尿1s喝50ml水并在最后一次选择后延长20min结束。
        1次  允许排尿0-10s（0-300ml），但每排1s需要排完立即喝50ml水
        1次 叠1张餐巾纸放在内裤里，控制少量排尿尿湿纸但是不能尿湿内裤。如果在厕所外进行尿湿内裤不需要惩罚，在厕所内进行，尿湿内裤则在最后一次选择后延长30min结束。
        1次 在60s内持续循环排尿与回憋，每次中断至少3s，完成15次奖励排出150ml，少于12次需要饮下300ml水，少于10次需要饮下300ml水并在最后一次选择后延长30min结束。
