昨天刷论坛的时候，脑海里忽然就浮想起这么多年的一些事情。

首先，哈哈总结一下自己这么多年的进步。

// 想起，第一次测容量时候的自己
那个时候刚上论坛还没多久。初中二年级假期的一个下午，趁家里人都出门，一个人躲在厕所里完成了一直以来的心愿和好奇。360ml，那是我已经很努力的数字。然后就是匆忙和紧张的清理现场，生怕家里人突然回来，等都做完了才松了一口气。再后来，初三结束的时候这个数字变成了 460ml，高一期中的时候好像是 480ml，高三结束之后已经是 740ml。现在大二，最近两年感觉都在 720ml 到 750ml 之间徘徊。

// 想起，小学第一次举手上厕所的自己
小学一年级的一堂美术课上，坐立不安的我举起了手。那时候也不懂什么羞怯，直接就说：“老师，我想小便”。“快去~”。我第一次在大家的注目中跑出去，有些尴尬却也觉得兴奋。等到了三年级，课堂纪律就严格了不少，我也开始学着忍耐。五年级以后，好像大家都不说“小便”了，改成“上厕所”，但即便如此，上课去厕所的人也没有之前那么多了。三四年级的班主任是数学老师，五六年级的班主任是语文老师，虽然都是拿过“优秀班主任”表彰的女老师，但我一点也不觉得她们可爱。三年级的一个中午，大家在班里聊得太开心的声音有点大，按她的话说，“在数学办公室都能听见你们吵吵”。那天下午要上的第一节课正巧是她的数学课，她便破门而入，班里瞬间安静下来。她坐在讲桌后面，让我们手背后坐好，不许动，不许下座位。全班就这么坐了半小时，先是午休下课铃响，再是第一节课上课铃响，可谁也不敢说话。然后，下午的数学课就开始了。我和很多人一样中午吃饭的时候都喝了汤，这个时候就只能忍耐。上课到一半的时候，我感觉快要失去控制了，就举起了手，“老师，我想上厕所”。正在这时，我斜前方的女生和她旁边的男生也都说要去厕所。老师没好气的说了声“不许去，都给我憋着”。我就这样度过了接下来煎熬的20分钟，确切的说是15分钟，因为在还剩5分钟的时候，我就已经控制不住自己的小便了裤子。我也不敢跟老师说，下了课，跑到厕所用纸巾使劲吸自己的秋裤外裤，希望不被看出来，由于外裤颜色深，也确实没看出来，可这是我第一次在小学尿裤子。到六年级的时候，拖堂已经很常见了，忍耐的状况也就越来越多。

之前专门写过自己小学的故事 （https://www.shireyishunjian.com/ ... read&tid=107763）

// 初中和高中
我在初中时候找到了论坛，像是打开了新世界的大门，那个时候好像有一篇很火的文章，主人公是小月。再后来，也不断有很火的作品出现，但是名字都记不太清了，好像有一个的主人公是小雨。那个时候，论坛上的区区好像已经在了，好像还有舒米勒什么的。初中大概算是一个明显的分界线，很多人对憋不住尿了裤子的记忆只停留在小学，而我不仅把它带到了初中，还意外地带到了高中。

初中的第一任班主任是一个脾气很不好的中年女老师，上学第一天就强调了课堂纪律：上课不允许去厕所是其中重要的一条。然而，开学刚刚五天的时候，还是有一个女生撞在了枪口上。班主任的语文课上，坐在我左前方的女生举起了手，这是开学以来第一个举手去厕所的同学。“你是不是小学没毕业啊？你下课干嘛去了？”那个女生估计是真的憋急了，很委屈地说自己憋不住了。“那也不行，下课再去”。女生坐下了，特别小声的骂了一句。我并不太清楚后来发生了什么，只知道这个老师一个月以后就离开了我们学校，可能是家长投诉教委了吧。再后来，我们就换了一个脾气还不错的语文老师，这个老师在去厕所这件事情上比较通融，想去厕所自己走出去就是了。

按说应该是没什么风险了，但我们学校有换班制度，根据每个月的月考成绩排座次，考得太差的同学就会被调出去。我所在的一班压力最大，经常有人出出进进。后来终于轮到了我。初二的第一次月考我就考了个128名，去了4班。4班算是个中游班级，普通班里的第一名，班主任是年级组长，教数学的，特别严厉，进入新班级的第一天我就体会到了。

“我们班转来一位新同学，XXX”。早自习的时候，老师带我进了班，并把我介绍给大家。她并没有说我是以前一班的，大概这好汉是不能提当年勇的。班里并没有我的空位，她就让我坐在第三排正中的位置，教室里都是一列一列的，没有同桌，据说是防止上课聊天。早自习跟第一节课上课之前，一般有5分钟的休息时间，在原来的班里，大家都是整理整理东西，喝口水，去去卫生间什么的。然而，早自习结束的铃声刚过，她就喊了“上课”，班长叫大家起立。一站起来，就觉得自己想去厕所，早饭喝了牛奶，一早上都没有去了呢。然而，好像真的上课了，没有任何预先通知的就进入了状态。我还是习惯性的悄悄往门口走，刚一起身她立刻叫住我，“XXX，你干嘛去呀？”，“上厕所”我小声说，班里的同学都抬起头看着，“上课呢知不知道啊？不要以为你以前是1班的就可以违反纪律，坐下”。我委屈的坐下，到新班级的第一节课就在憋尿中度过。从此以后的一年，我在学校经常憋尿，甚至有了一个第二节课以后很少喝水的习惯，因为第三节下课就要课间操，回来可能就会直接上第四节课，直到下课才能去厕所。然而，这个严厉的女老师大概也是让我开始发现乐趣所在的人。在初二的后半学期，我甚至会寻找连堂两节甚至三节的数学课，在上课之前喝完整整一大杯水，然后故意憋尿，寻找一种特别的紧张和乐趣。初三的时候，我又回到1班，但我们班的数学还是她教。我还是会下课不去厕所，然后等待那份急迫慢慢地到来。

我的初中和高中在同一所学校，总体来说都有很好说话的老师，也都有很严的老师，比如我之前写过的高一那次（https://www.shireyishunjian.com/ ... read&tid=108605），这是高中时候比较深刻的一次了。因为晚自习不让上厕所，所以忍耐的次数其实并不少见，有很多次已经都差点要失禁，要么是关键时刻得到了老师允许，要么是一咬牙就坚持了下来。但无论如何，在高中的时候，直接举手去厕所的情况已经很少了，一般都是等老师过来悄悄说。高二以后在文科班，因为几乎没有男生所以大家就更放松一些，常能听到一下课就有人喊“憋死我了，快陪我去厕所”这样的话。

//课堂以外
这些年，课堂以外的憋尿其实挺多的，主要是各种活动还有各种学生干部的工作，以后有空就写一写。

最近想要憋一次，找找当年课堂上那种美好的感觉。
