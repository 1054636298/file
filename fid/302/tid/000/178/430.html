这是我第一次写计划/任务，有什么不足的求补充
加水可以按你自己的喜好加，作者我觉得加那么多水有点不真实
任务，挑战，计划，惩罚，游戏
重中之重：只是玩玩而已，不用太认真，身体为重，感觉不对劲或者疼痛一定要停。更困难模式以后根本不是人能做的。注意：锻炼的阶段要量力而行，我也不知道bn的时候锻炼腹肌会不会对身体有害，觉得自己坚持不住了就停。
一共要选三个难度，分别是第一，第二和第三

第一
简单模式 - 游戏前排空
普通模式 - 游戏前不排空
困难模式 - 游戏前不排空+全程分开腿
更困难模式 - 游戏前不排空+全程休息时间分开腿+一个勒紧腰带
困难再加更困难模式 - 游戏前不排空+全程休息时间分开腿+一个勒紧腰带+全程休息时间蹲姿/踮脚
地狱模式 - 游戏前不排空+全程休息时间分开腿+一个勒紧腰带+全程休息时间蹲姿/踮脚+惩罚，挑战翻倍

第二
简单 - 正常喝水
普通 - 喝水量x2
困难 - 喝水量x3
地狱 - 喝水量x3+利尿剂

第三
简单 - 第一难度惩罚&挑战
普通 - 第二难度惩罚&挑战
困难 - 第三难度惩罚&挑战
地狱 - 所有难度的惩罚&挑战都做一遍
（不建议地狱模式因为挑战很相近，会无聊）

加成（简单以上要有难度加成）
掷骰子：
1：全程不许用手捂
2：全程裸体
3：每次惩罚再掷一次骰子（执行两次惩罚）
4：硬币获得除以二
5： 用夹子夹乳头和胯部，直到游戏结束+只能在商店买排尿时间
6：游戏期间排尿时间除以1.5

游戏商店（得到硬币来这里兑换）
跳过等待时间 - 5硬币
喝水赦免 - 30硬币
惩罚赦免 - 100硬币
加强惩罚赦免 - 120硬币
排尿2秒 - 15硬币
排尿4秒 - 25硬币
排尿6秒 - 45硬币
排尿8秒 - 85硬币
排尿10秒 - 160硬币
排尿20秒 - 300硬币
排完 - 500硬币，结束游戏，硬币清空
注：硬币是可以累积的，你这次不用的话下一次游戏还会有你上一次的硬币

请确认已经选好难度

注：在下面你可能会看见^这个符号。有^的是指令，因为作者太懒不想把描述写第二次所以只要有^的地方就要做跟上一次^一样的行动。
比如（只是比如而已，一切以下面的游戏开始之后为准）
^喝水 - 喝60ml水
等待十分钟
^喝水
虽然第二次的指令（^喝水）没有给出具体的解释，但是因为他是指令，所以要做跟上一次一样的行动，所以概括一下就是：
喝60ml水
等待十分钟
喝60ml水
不只是前面的符号要匹配，后面的文字也要匹配。比如
^喝水 和 ^等待 是不一样的，尽管它们都有^。（好吧我突然感觉不那么懒了我还是都写下去吧）


游戏开始
（剧情可能有点智障但是不要在意那些细节）


【第一阶段】 - 准备阶段

你醒了个大早，因为你听说早上喝水对身体好，所以你打算烧一壶水。
^喝水 - 投掷骰子，喝骰子点数乘以100的水（ml），喝水后硬币+10

突然发现你好像烧多了，又不想浪费水，所以打算分几次把水喝完。
投掷骰子，骰子数等于你烧多了多少水，也就是 S

^等待 - 你的胃现在还装不下那么多水，等待十分钟
十分钟之后之后继续^喝水，喝完之后S-1，然后继续^等待并重复这个流程，直到S = 0，你又烧了一壶水，等了十分钟，进入第二阶段。

‘第一阶段大致流程示例’：
投骰子喝水，投了个3，喝三百ml。
投骰子，投了个4，多出来四份水，所以说还要喝四次水

等待十分钟
投骰子喝水，投了个2，喝二百ml。

等待十分钟
投骰子喝水，投了个4，喝四百ml。

等待十分钟
投骰子喝水，投了个6，喝六百ml。

等待十分钟
投骰子喝水，投了个1，喝一百ml。
没水了，等十分钟，进入下一阶段



【第二阶段】 - 铺垫阶段

你到厕所，像往常一样想上个晨厕，但在你就要上的时候你记了起来昨晚因为觉得马桶太丑，把清洁剂倒在了马桶里，在使用册子上说要看到白色的清洁剂变的深黄才可以洗，要不然不会完全清洁马桶，会浪费了那一瓶清洁剂。想到那加强版清洁剂的高昂价格，你决定先不尿。

-----挑战-----
简单 - 模拟尿尿（尿尿但是快要尿出去的时候憋回去，五次
普通 - 边按小腹边模拟尿尿，十次
困难 - 边按小腹边尿：尿一秒然后憋回去，十五次

^等待（这次五分钟）
你走出厕所，躺到了床上，钻到了被子里。天气冷使得你不停的往里钻。

-----挑战-----
简单 - 在床上把重心放在小腹上，用床压小腹，使劲压。2分钟
普通 - 压5分钟+打屁股10下
困难 - 压10分钟+打屁股30下

之后加水200，等十分钟，再加水两百，阶段完成获得硬币30，进入下一阶段



【第三阶段】 - 无聊阶段

你现在感觉有点无聊，想要找点乐子
但是你暂时还没有想到找什么乐子，所以你打算发呆
-----挑战-----
简单 - 坐在椅子上放松膀胱和下肢，三分钟
普通 - 坐在椅子上，脚放在椅子上，双腿分开放松膀胱和下肢，四分钟
困难 - 坐在椅子上，小腹上放重物，放松膀胱和下肢，五分钟
挑战完成后硬币加25
之后加水200，等十分钟，再加水两百，进入下一阶段
^等待（这次五分钟）

你想用一会手机，但是自从上次你看了一篇关于防贼的文章之后你就把手机放到了柜子顶，你垫脚跳几下但是还是够不到
-----挑战-----
简单 - 踮脚跳20次
普通 - 踮脚开合跳25次
困难 - 踮脚跳蹲起20次
挑战完成后硬币加15
^等待（这次五分钟）

你打算把一个椅子搬到柜子旁，但是因为你忘了吃早饭，有点没力气，所以搬不动。
-----挑战-----
简单 - 深蹲15次
普通 - 举着椅子深蹲10次（我试了一下有点难，量力而行吧）
困难 - 单腿不举任何东西深蹲10次，每次之后换腿，然后按小腹一下
挑战完成后硬币加15

之后加水200，等十分钟，再加水两百，阶段完成获得硬币30，进入下一阶段


【第四阶段】 - 运动阶段

你在无聊刷抖音的时候看到“每天运动，强身健体，还有助于排毒！”
所以你打算试一试
-----挑战-----
第一小阶段 - 热身
简单 - 跪姿俯卧撑10个，踮脚站立一分钟，坐位体前屈（双手要触碰到脚腕）20秒
普通/困难 - 摸膝盖20次，卷腹30次，仰卧起坐10个
小阶段完成后硬币加20
喝水100ml，等待5分钟进入下一小阶段

第二小阶段 - 腿部锻炼
简单/普通 - 跳绳10个，爬楼梯20个台阶，踮脚侧身高抬腿25次
困难 - 鸭子步30步，蛙跳10个，深蹲15次
小阶段完成后硬币加20
喝水100ml，等待5分钟进入下一小阶段

第三小阶段 - 腹部训练
https://www.jianshu.com/p/e041d7818665
有一些动作后面标注（J站）的是从上面网站搬运的，一般都不是一些耳熟能详的动作。我会标注出来号码，不知道动作的人可以上上面的网站查询

简单 - 臀桥25秒，仰卧起坐15次，登山10次（J站中级第七个）
普通 - 仰卧抬腿15次，空中自行车三十次，平板支撑二十秒
困难 - 简化版俄罗斯转体15次（一次一整个半圈），仰卧对角交替收膝15次（J站初级难度第五个），仰卧屈膝卷腹10次（J站初级难度第十，中级难度第十五个（我也不知道我为什么会有两个，可能作者发重复了））
小阶段完成后硬币加20

之后加水200，等十分钟，再加水两百，阶段完成获得硬币30，进入下一阶段



【第五阶段】 - bn阶段

你现在觉得有点憋了，但是你不能去上厕所。你打算尝试一些奇怪的姿势来打发时间。
-----挑战-----
简单 - 蹲下15秒
普通 - 跪下，左右膝盖与肩同宽，脚掌并拢，双手放在后脑勺，挺直身体五分钟
困难 - 土下座20秒
等待五分钟

-----挑战-----
简单 - 俯卧在床上，小腹下垫一个枕头十分钟
普通 - 仰卧在床上，小腹上一本书十分钟
困难 - 俯卧在床上，背上压个重物八分钟


之后加水200，等十分钟，再加水两百，阶段完成获得硬币40，进入下一阶段



【惩罚阶段】
如果挑战失败或者中途失禁超过一秒则进入惩罚阶段
-----惩罚-----
掷骰子
1，劈叉15秒
2，牙刷刷尿道15次
3，数据线打屁股35下
4，按小腹15下
5，俯卧平躺30秒
6，绕着自己的房间爬行五圈

之后加水200，等十分钟，再加水两百，进入失禁前的阶段



【加强版惩罚阶段】
如果在惩罚阶段失败或失禁则进入加强版惩罚阶段
-----加强版惩罚-----
掷骰子决定：
1，排尿回憋15次，不许尿出来
2，腹背挺身（重心放在小腹）10次
3，模拟游泳（重心放在小腹）15秒
4，侧卧腹斜肌卷腹30秒（J站中等第八）
5，仰卧交替抬腿25次（J站中等难度第十一）
6，仰卧屈膝提髋10次（J站高级难度第一）

之后加水200，进入失禁前的阶段



【特殊阶段】 - 打工阶段（在等待的时候打工）
看着商店里面的东西眼馋？想买不能买？想要硬币？打工阶段来硬币快！
-----打工阶段挑战-----（还需要几个职业）***

迎宾小姐/先生：挑战完成后加30硬币
简单 - 作为一个迎宾人，客人来了之后要有礼仪。鞠躬15下。
普通 - 踮起脚尖鞠躬20下
困难 - 双腿分开踮起脚尖鞠躬30下
每一次打工阶段完成后不需要等待。

健身教练：
（因为是个教练所以在室外挑战完成硬币加倍，嘿嘿嘿！）
简单 - 空中自行车40个，完成后硬币加20
普通 - 仰卧，在小腹上放一个书之后卷腹20个，完成后硬币加40
困难 - 俯卧，在小腹下放一个很高的物体做跪姿俯卧撑30个，完成后硬币加五十

走秀模特：（em我觉得之前的有点清口了，现在来点重口的）
简单 - 系紧一个腰带，要是已经在难度里选择了腰带的话就再系的紧一点，之后绕着房间走5圈，完成后硬币加三十
普通 - 内裤里放跳蛋，男生的话边走边撸，记住撸的时候一定要每一次都砸到小腹，绕房间7圈，完成后硬币加40
困难 - 踮脚，走螃蟹步（横着走），绕房间10圈，每一圈完之后按小腹五下，完成后硬币加70

罪犯：
简单 - 用绳索束缚双手和双脚，跳着绕着房间走5圈，完成后获得硬币30
普通 - 用绳索束缚踮着脚，跳着绕着房间走8圈，完成后获得硬币50
困难 - 用绳索束缚着双手双脚和小腹，踮着脚，跳着绕着房间走10圈，最好弄成一跳小腹的绳索就会收紧的那种。完成后获得硬币80

暴露狂：
简单 - 出门跑三百米，完成后获得硬币40
或
出门带着跳蛋跑三百米，完成后获得硬币50
普通 - 出门做一套运动阶段--第一小阶段--简单难度的三个运动，完成后获得硬币40
或
裸体暴露在窗前2分钟，完成后获得硬币40
困难 - 出门到一个僻静的地方edge，完成后获得硬币60
或
穿着一条尿湿的裤子到楼道里往上一口气爬5楼，完成后获得硬币30



【最后阶段】 - 结束阶段
你倒在马桶里的清洁剂终于有了变黄的迹象。
注：在结束阶段失败直接去加强版惩罚阶段
掷骰子决定：
1:
简单 - 听流水声15秒，凉水洗手10秒，游戏结束，获得10硬币
普通 - 听流水声15秒，游戏结束，获得20硬币
困难 - 按小腹10次，听流水声15秒，凉水洗手10秒，游戏结束，获得30硬币
2:
简单 - 再忍耐20分钟，然后释放，游戏结束，获得30硬币
普通 - 再忍耐40分钟，允许用手捂，然后释放，游戏结束，获得40硬币
困难 - 再忍耐80分钟，允许夹腿和用手捂，然后释放，游戏结束，获得50硬币
3:
简单 - edge1次，然后释放，游戏结束，获得20硬币
普通 - edge2次，然后释放，游戏结束，获得40硬币
困难 - edge4次，然后释放，游戏结束，获得70硬币
4:
简单 - 仰卧直腿卷腹触足卷腹10次（J站中等难度第十四），然后释放，游戏结束，获得40硬币
普通 - 仰卧直腿卷腹触足卷腹20次（J站中等难度第十四），然后释放，游戏结束，获得60硬币
困难 - 仰卧直腿卷腹触足卷腹30次（J站中等难度第十四），然后释放，游戏结束，获得100硬币
5:
立即释放，游戏结束但清空所有硬币
6:
你有两个选择：
1，从头再来，获得200硬币
2，憋着直到完全失禁，获得300硬币
