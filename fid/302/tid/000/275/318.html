科学扩容计划喝水量计算公式（男女通用），扩容计划在主页
该公式由论坛一个大佬提供，我做成了python代码方便计算。
会用的自己用，不会用的可以从主页加我帮你们算
代码如下：#号后面是调试用，可直接删除；复制可直接使用，男女通用。
#健康喝水计算系统
#(毫升) = 0.4220 x (公斤体重^0.425) x (厘米身高^0.725) / (年龄^0.203) + 86.5 x (摄氏度气温/25)^3.322
#(注意: 符号^表示乘方，例如100^2=100 x 100)
a = int(input("体重(公斤)"))
b = int(input("身高（厘米）"))
c = int(input("年龄"))
d = int(input("温度（摄氏度）"))
e = (input("性别(男\女)"))
if e == "女":
    f1 = 0.4220 * (a ** 0.425) * (b ** 0.725) / (c ** 0.203) + 86.5 * (d / 25) ** 3.322
    print(round(f1))
elif e == "男":
    f2 = 0.5686*(a**0.425)*(b**0.725)/(c**0.203)+103.5*(d/25)**3.322
    print(round(f2))
else:
    print("请输入正确的性别")
