无法排尿的女儿与担心她贞操的妈妈

1
那是我担任住院总期间，秋天的傍晚，门诊已经结束，其他医生都已经下班，值班的规培医生去食堂吃饭了，当时科里就剩下我个人。突然接到急诊打来的电话，说有个12岁的女孩排尿困难，请我下去。白大褂口袋里装上导尿装备，赶到诊区走廊，就能听见妈妈在训斥女儿：“让你平常别老是憋尿，你就没听过，这下好了，陪你来看病，要耽误你弟和你爸吃饭了”走到诊室门口，我看到那个小姑娘，捂着小腹，扶着墙艰难地站着。面色因痛苦显得略有苍白。“医生，我们小朋友已经5个多小时没有尿过尿了，1个多小时之前我以为是她喝水太少，还给她灌了两大碗水，可不知为啥，他小便就是下不来，你给看看是不是平常憋尿憋得！”说着又瞪了孩子一眼。我让孩子躺倒检查床上，小腹部隆起，触诊张力很大，压痛阳性拒按。“这种情况需要马上导尿”我跟家长说道，交代了一下导尿的注意事项和可能发生的不良反应，我让孩子脱下内裤，张开双腿，随即，我也明白了小姑娘尿潴留最可能的原因-孩子的阴道口并不存在，取而代之的是隆起的，完全封闭的处女膜，因为阴道内积攒的经血无法排出，导致处女膜看起来发出诡异的蓝紫色。处女膜是位于阴道外口和会阴的交界处的膜性组织，正常处女膜分为有孔型、半月型、筛状、隔状、微孔型。如完全无孔隙，则为处女膜闭锁，是女性生殖器官发育异常中较常见的类型。处女膜闭锁的孩子不手术的话月经血是无法排出的。所以自从孩子月经初潮以来，每次月经产生的经血都会积聚起来，一开始是积聚在阴道中，随后是子宫，最后是输卵管，在这个过程中，孩子会出现越来越明显的下腹部疼痛，还有就是大小便的问题，因为增大的阴道会向前压迫尿道，向后压迫直肠。今天接诊的这个孩子就是被极度扩张的阴道压迫到了尿道，导致尿道无法打开，从而引起尿液无法排出。“孩子的尿潴留是由于处女膜闭锁导致的，你们孩子这么晚没来月经，为什么都不来医院看一下”我对这个孩子父母的粗心有些震惊。“哎呦医生，我平常要照顾一家老小，哪有空管这些事情！”，“你怎么回事？平常有不舒服也不跟我们说！”孩子的妈妈仿佛是个负能量集合体，一般对我抱怨，一边数落孩子。小姑娘就只管流眼泪，也不说辩解的话。导尿管顺利进入膀胱，随即尿液便快速流出，几分钟的时间，尿液就多达1000毫升，要知道，一个成年人的正常膀胱容量是500-800ml。   

2
我匆忙把导尿管折起来阻止尿液流出，要知道，对于严重尿潴留的病人，如果膀胱快速排空，会导致盆腔内压力骤降，血液暂时滞留从而引起低血压。随后用了半小时时间，我开开关关缓慢放出了共2000多毫升尿液。孩子的表情终于从痛苦逐渐变为忧伤。随后我带着他们去超声室做了一下妇科超声检查。超声可以看到因为经血无法排出，整个阴道和子宫都被积攒的经血塞得满满当当。在这个过程中，孩子的妈妈仍然在不停地问我什么时候能拔导尿管回家。似乎家里的儿子和丈夫比现在生病的女儿还要重要。根据孩子的症状和超声检查，已经基本可以排出其他疾病，确认是处女膜闭锁了。“你们今天怕是走不了了，小朋友这个情况是典型的处女膜闭锁，需要急诊手术，排出经血，不然从近期说，孩子脱离不了导尿管，从远期说，孩子的月经血越积越多，会导致腹痛和不育。”说完之后，妈妈的怨气又爆发了，“我怎么这么倒霉，生了你这个赔钱货！”。小姑娘全程一言不发，就是在抹眼泪，看得人心疼。“这位妈妈，请注意你的言辞，孩子的病是先天性问题，不是她的错。”我实在是听不下去了，打断了妈妈的抱怨。随后就是办理住院，办好后，我开始跟家长做术前谈话，您孩子的这种情况，需要把闭锁的处女膜切开，不然经血流不出来会导致一系列问题。手术过程主要就是把处女膜切开，然后把残存的经血吸引出来，随后把多余的处女膜切除。因为这种病的特点就是孩子的处女膜比正常人要厚，所以术后有可能会出现阴道口挛缩的情况，当然可能性不太大，我们术中也会注意尽量把增厚的处女膜切干净一点。说到这，妈妈突然开始激动：“医生，你要是把我们家孩子处女膜全切掉，以后是不是就不能算是处女了？是不是就不会有处女血了哇？能不能给我们留一点处女膜组织，这样以后她嫁人，也好向婆家交代。”听到这些话，我简直不敢相信自己的耳朵，都2024年了，在上海还有这样保守的母亲。我们的手术目的首先是最大限度解除孩子的病痛，您的要求可能会加重孩子以后阴道口挛缩的可能性，我们没法按您的要求做。不过我们会尽量注意孩子手术切口的美观。话说到这，妈妈似乎还是心有不甘，不停地说着要让我少切一点。旁边的规培医生听不下去了，说道，你们是来治病，不是来整形的！。孩子妈瞥了规培的一眼，不再多说什么。毕竟这个手术还存在着尿道损伤，直肠损伤和感染的风险。

3
好不容易终于把术前谈话的字签好，随后我们又麻利的完成了术前常规检查和术前准备，终于在3小时孩子禁食时间充足后上了手术台，麻醉后消毒铺巾，注意尿道口和肛门位置，先在处女膜表面做一个X形切口，随后就能看到大量的又黑又浓的月经血涌出，最终我们从这个小姑娘的阴道里吸出了1500毫升左右的经血。要知道经血残留在子宫里是会出现逐渐加重的腹痛和痛经的，同时还会伴随尿频和便秘。有这么一个妈妈和家庭，我很难想象这个女孩将来的人生会是什么样子。完整切除处女膜，留置引流，结束手术。
第二天查房，伤口无明显出血，拔出导尿及引流，患儿排尿正常，下午出院。

（完）

最后也希望小女孩能尽早摆脱病痛和保守家庭的困扰，祝她好运。

其实倒也不会有多大，如果膀胱是个完美的球体的话，2,000 ml 的球只比 1,000 ml 的球的直径多 3 cm（是直径，不是半径哦）。当然，膀胱不是完美的球，而是一个“扁球”，甚至会被拉伸成“长球”状——但这样来说在纯粹的“凸起高度”上可能就更低了。

我们常常忘记了容积和表面积、和半径不是成正比的。一个完美球体的体积如果增加 x 倍，那么它的表面积只会增加 x 的三分之二次方倍，它的直径（或者半径）只会增加 x 的三分之一次方（也就是开三次根号）倍。

也就是说，如果一个体积为 500 cm³ 的球，如果每次使其表面积增大约 25%（精确来说是 25.1565%），那么它的体积的增大将是很夸张的，取前七次增大后的体积如下：
500 -> 700 -> 980 -> 1372 -> 1921 -> 2689 -> 3765 (cm³)
而与此同时，其直径的增加幅度却是很少的：
9.85 -> 11.02 -> 12.32 -> 13.79 -> 15.42 -> 17.25 -> 19.30 (cm)

如果将表面积的增大从固定比例改为固定实际增加量，那么以从 500 cm³ 增到 700 cm³ 的这次的增加量（76.6 cm²）来算，如果固定增加 7 次，其所对应的体积如下：
500 -> 700 -> 921 -> 1162 -> 1420 -> 1696 -> 1987 (cm³)
而其直径的增加如下：
9.85 -> 11.02 -> 12.07 -> 13.04 -> 13.95 -> 14.80 -> 15.60 (cm)

举这两个例子是想说明，体积（容积）的增大，与直径（也即换到膀胱这个问题上大致可以对应“凸起高度”）和表面积（换到膀胱问题上就是膀胱壁被拉伸的程度）的增大不是成正比的。以第二例来看，当容积增大了近 4 倍后，表面积才增大了 2.5 倍，直径才增大了 1.58 倍。

当然膀胱不是完美球体，所以此处仅仅是通过一个简单的例子来使我们认识到，人们可能往往会以为：2,000 ml 的容器，那得比 1,000 ml 的容器大多少啊，得大整整一倍吧？！其实不然，从视觉上来看，只是大了大概 26%（从直径 12.4 cm 长到直径 15.6 cm）。

那么 12 岁的膀胱能不能够盛下 2,000 ml 的液体呢？也是完全没问题的。

论坛上的网友 @mnbvcxzq 曾经分享过一个病例（https://www.shireyishunjian.com/ ... read&tid=158420），里面讲的是一个 5 岁的小女孩，因为基因突变，导致无法感知到排尿的欲望，因此膀胱给憋到了能容下 3 升之多。这个病例里的小女孩是 2005 年 3 月 5 日出生，到 2009 年 11 月去看泌尿科的时候（因为这个时候她已经发生了肾反流，日期在其他篇目的日志中有提到），已经到了这个地步了。也就是大概才 4.6 岁左右。4.6 岁的孩子，按照那个公式来算的话，正常的膀胱容量才 200 ml 而已。然而从出生到四岁半，也就四年的病程，就让一个婴孩的膀胱扩张到了惊人的 3 升。

因此可见，当人的膀胱因为某种疾病长期处于憋尿状态时，容量是会迅速增加的。在楼主分享的这个病例中，虽然描述上写的是五小时未排尿，但考虑这个案例的情况，其实这个女孩子应该有挺长一段时间都排尿困难了，可能之前能排的时候也无法排净，一直处于有尿液残留且可能越累积越多，直到这次就诊时彻底给“憋死”了一点儿都无法排出。因此并不是一天之间就憋到 2,000 ml 的，而是应该有一个过程。

人在某种疾病状态下膀胱可以撑到非常大，英国有一个老人在长达 15 年的病程中将膀胱给憋到了可以容纳 16 升的尿液而不破（https://www.eurorad.org/case/18366），这个大膀胱的“三围”达到了 21.3 cm x 26.8 cm x 30.2 cm，其中 26.8 cm 这个是向前（外）凸的长度，如果认为盆腔里可以容下 10 cm 左右的长度的话，也就是平躺时大概会凸出大概 17 cm 左右。
