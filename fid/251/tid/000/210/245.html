使用方法:

1. 点击下载

2. 运行。如果 Windows 报告 “阻止了无法识别的应用启动”，点 更多信息 > 仍要运行。
    Windows 报告他有风险是因为，任何一个运行人数不够多的新程序，都是“无法识别的”。这个程序七海检查过源代码没有安全问题。

3. 访问 https://www.shireyishunjian.com/ 开始冲浪。
    推荐使用 Chrome 或 Edge 浏览器。

credit: @codetector
